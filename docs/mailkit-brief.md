# MailKit — бриф нового SaaS (рабочее название)

Статус: идея, бриф для оценки. Не в работе. Зафиксировано 2026-04-20.

---

## 1. Суть продукта одной фразой

Подключи домен и Gmail — через 5 минут у тебя работает `hello@твой-домен.kz`: приём писем в Gmail + отправка из Gmail с этого адреса, без ручной возни с DNS, SMTP и Send-As.

## 2. Проблема

Каждый соло-фаундер / indie hacker при запуске нового проекта проходит один и тот же ад:
- Cloudflare Email Routing — добавить домен, создать DNS-записи, включить routing
- Brevo / Resend / SES — верификация домена, DKIM-ключи, SMTP-credentials
- Gmail Settings → Accounts → Send mail as — вручную SMTP, код верификации
- Gmail Signature — собрать, вставить, привязать к нужному ящику

30+ минут даже если ты делал это 5 раз. Три разных UI, три терминологии, каждый периодически меняется. В голове не автоматизируется — всегда приходится переспрашивать ИИ или гуглить.

## 3. ICP (целевой клиент)

**Serial soloprenuer / indie hacker** с 3-10 активными проектами.
- Технически подкован (знает что такое DNS, SMTP, OAuth)
- Ценит своё время сильно выше $50/час
- Уже платит за десятки SaaS-подписок, устал от ежемесячной возни
- Запускает новые проекты регулярно (раз в 1-3 месяца)
- Gmail — основной рабочий ящик, переезжать на Zoho/Workspace не хочет

Вторичный ICP: небольшие агентства и freelancer-ы, которые делают лендинги клиентам «под ключ» и включают настройку почты как часть сервиса.

## 4. Решение — флоу за 5 минут

1. **Регистрация** — OAuth Google (один клик)
2. **Connect Cloudflare** — OAuth или API token (один клик)
3. **Выбрать домен** из списка Cloudflare-зон
4. **Ввести желаемый адрес** — `hello@mydomain.kz`
5. **Platform делает всё за кадром:**
   - Создаёт Email Routing на Cloudflare
   - Создаёт домен в Brevo/Resend (выбор провайдера на нашей стороне)
   - Добавляет DNS-записи через Cloudflare API (MX, SPF, DKIM, DMARC)
   - Ждёт propagation (обычно < 1 мин для Cloudflare)
   - Через Gmail API добавляет Send-As с SMTP-credentials
   - Ловит код верификации из inbox через Gmail API → автоподставляет
   - Ставит подпись (заранее настроенный шаблон или кастомный)
6. **Готово.** Юзер видит «Отправьте тестовое письмо» → проверяет работу.

**Ручных действий:** 0 после 2 OAuth-кликов на старте.

## 5. Бизнес-модель

### Основная: Pay-per-setup

| Что | Цена |
|---|---|
| Настройка одного email-адреса на домене | **$5** |
| Bundle: 3 ящика на одном домене | **$12** (экономия $3) |
| Bundle: домен + 5 ящиков + группа-алиасы | **$25** |

Логика: соло-фаундер видит стоимость как эквивалент «2 кофе», не думает, жмёт покупку. Сравнение: 30 минут его времени ≈ $25-50, значит $5 — очевидная экономия.

### Upsell №1: Deliverability monitoring — $3/мес за домен

- Проверка каждые 6 часов: попадают ли тестовые письма в inbox Gmail/Outlook/Yahoo
- DMARC aggregate reports — обработка и визуализация (кто спуфит твой домен)
- Alerts в Telegram/email если SPF/DKIM/DMARC сломались или репутация упала
- Отчёт в дашборде: inbox rate, spam rate, топ-ошибки

Целевая аудитория апсела: те у кого пошёл трафик писем (newsletter, outreach, транзакционные).

### Upsell №2: Multi-mailbox / multi-domain bulk — $15/мес

- Безлимит настройки новых ящиков и доменов без оплаты за каждый
- Управление всеми ящиками из одного дашборда
- «Клонировать настройку» с одного проекта на другой в 2 клика

Для агентств и фаундеров с 10+ проектами.

### Upsell №3: Custom email signature editor — $10 one-time

- Визуальный редактор подписи
- Шаблоны (минималистичный, корпоративный, с градиентом, с фото)
- Автоматическая установка в Gmail через API
- Множественные подписи (для разных ящиков)

### Upsell №4: White-label для агентств — $49/мес

- Кастомный домен дашборда (setup.agency.com)
- Лого клиента в email-подписях
- Billing через агентство, клиенты не видят MailKit
- API для автоматизации onboarding клиентов агентства

## 6. Unit-экономика

### Расходы на одну настройку

- Brevo / Resend / SES — базовый план бесплатный или ~$0/ящик при малом объёме
- Cloudflare Email Routing — бесплатно
- Наши сервера / API calls — ~$0.10-0.20 за настройку

**Gross margin на pay-per-setup:** ~95-98% ($4.80 из $5)

### Потенциал выручки

Консервативный сценарий:
- 500 платящих пользователей на первый год
- В среднем: 2 setup в год + 30% конверсия в monitoring subscription
- 500 × 2 × $5 = $5,000 (setup)
- 500 × 0.3 × $3 × 12 = $5,400 (monitoring)
- Итого: ~$10K/год Year 1

Оптимистичный:
- 5,000 платящих к концу Year 2
- 3 setup в среднем + 40% в monitoring + 10% в multi-mailbox plan
- 5,000 × 3 × $5 = $75,000 (setup)
- 5,000 × 0.4 × $3 × 12 = $72,000 (monitoring)
- 5,000 × 0.1 × $15 × 12 = $90,000 (multi-mailbox)
- Итого: ~$237K ARR

Для соло-фаундера с минимальными затратами (Vercel + Supabase + Brevo-пропуск ~$100/мес) — хороший бизнес.

## 7. Конкуренты (research 2026-04)

### Direct (зарубежные)

| Сервис | Что делает | Цена | Что не закрывает |
|---|---|---|---|
| **Forward Email** | inbound forward + опционально outbound SMTP | $3-9/мес подписка | Не автоматизирует Gmail Send-As, DNS пишется вручную |
| **ImprovMX** | forwarding + SMTP для Gmail Send-As | $2.50/мес подписка | Step-by-step guide, всё равно ручные клики |
| **Maildoso / Infraforge / Mailforge** | автонастройка SPF/DKIM/DMARC для cold email | $100-733/мес (32-400 ящиков) | Cold-email агентства, не inde hackers; свои домены, не твои |
| **Smartlead / Instantly** | то же что Maildoso, но с outreach-платформой | $100+/мес | Не для personal email |
| **Resend Cloudflare Domain Connect** | один клик DNS на Cloudflare для outbound | per-send pricing | Только отправка; нет приёма, нет Gmail wiring |
| **MailerSend / MXroute** | SMTP-провайдеры | $1-9/мес | Ручной DNS, нет Gmail-интеграции |
| **Zoho Mail / Google Workspace** | свой почтовый ящик + quick-setup | $0-18/user/мес | Заменяют Gmail, не дополняют его |
| **EasyDMARC / DMARCLY** | DMARC-wizards | $20-100/мес | Только DMARC, нет SMTP/inbound/Gmail |

**Вывод:** никто не делает полный цикл «Cloudflare + third-party SMTP + Gmail Send-As + подпись» в одном OAuth-флоу за $5. Никто не работает по pay-per-setup модели.

### СНГ рынок

- **Яндекс 360 для бизнеса** — wizard для своего ящика, делегирование NS, 290₽/юзер/мес. Не для Gmail-юзеров.
- **Mail.ru для бизнеса / VK WorkSpace** — аналогично.
- **Нет СНГ-сервиса**, который автоматизирует Cloudflare + Gmail + любой SMTP.
- Для Казахстана есть дополнительный нюанс: часть юзеров не имеет доступа к платежам на РФ-сервисы → ниша ещё уже.

### Signal: Fiverr gig «email setup $10 one-time» существует
Люди реально платят за ручную настройку. Productized версия этого — и есть MailKit.

## 8. Позиционирование — уникальный клин

1. **Pay-per-setup, не подписка.** Единственный в категории.
2. **Gmail-centric.** Все альтернативы либо заменяют Gmail, либо для cold-email. MailKit «за» Gmail-лояльного юзера.
3. **Indie-hacker ICP.** Cold-email платформы обслуживают нужный объём доменов, но не тот use case.
4. **Cross-vendor orchestration.** Только мы оркестрируем Cloudflare + SMTP-провайдера + Gmail одновременно.
5. **Опциональная подписка** — не навязываем, продаём отдельно тем кому реально нужен мониторинг.

## 10. Риски и открытые вопросы

- **Gmail API ограничения** — `sendAs.verify` требует код из письма. Нужно протестировать на 10+ ящиках что автоматический перехват работает стабильно.
- **Cloudflare Terms** — коммерческая автоматизация через их API должна быть разрешена (скорее всего ок, но проверить).
- **Brevo/Resend terms** — могут запрещать «резелл». Решение: белый партнёрский аккаунт или несколько независимых (наш юзер → свой SMTP-аккаунт через нашу автоматизацию от его имени).
- **Google OAuth verification** — Gmail API требует verified OAuth app. 4-6 недель на верификацию, нужно заложить в timeline MVP.
- **Poaching** — Cloudflare или Resend могут встроить эту фичу сами. Митигация: быстрый go-to-market, захват бренда, расширение на мониторинг/multi-provider, где они не смогут (vendor-neutral).

## 11. MVP scope

Версия 0.1 для launch:
- Только Cloudflare + Brevo + Gmail (один стек, самый proven путь)
- Один язык (английский, rest-world market сильнее)
- Один флоу: create → verify → done
- Без monitoring (отложить на апсел после 50 платящих)
- Stripe + Lemon Squeezy для оплаты
- Landing + 5-минутное demo-видео

**Estimated time to MVP:** 3-4 недели соло с Claude Code.
