# Vibeaudit — бесплатный аудит скорости сайта

Лид-магнит для агентства Vibecraft. Страница /audit на vibecraft.kz.

---

## 1. Обзор идеи

Vibeaudit — бесплатный онлайн-инструмент на странице vibecraft.kz/audit. Клиент вводит URL → получает на экране отчет о скорости сайта с расшифровкой проблем на русском языке → видит CTA «Закажите исправление через Vibecraft» с промокодом.

### Зачем это нужно

Привлечение целевых лидов в агентство Vibecraft. Человек, который проверяет скорость своего сайта — уже осознает проблему. Ему нужно только увидеть масштаб проблемы и получить конкретного исполнителя. Vibeaudit делает оба шага за один визит.

### Почему бесплатно

Любой может прогнать PSI бесплатно и попросить ChatGPT интерпретировать результаты. Платить за то же самое бессмысленно. Ценность Vibeaudit — не в самом отчете, а в связке «диагностика → исполнитель». Инструмент бесплатный, деньги приходят через Vibecraft.

### Что это НЕ является

- Не SaaS-продукт. Нет подписок, нет личного кабинета, нет платных тарифов.
- Не конкурент PSI/GTmetrix. Мы используем PSI как источник данных, не конкурируем.
- Не отдельный проект. Это одна страница внутри существующего vibecraft.kz.

## 2. User flow

```
1. Клиент заходит на vibecraft.kz/audit
2. Вводит URL сайта + email
3. Нажимает «Проверить»
4. Индикатор загрузки 10-30 секунд (пока PSI API прогоняет)
5. На экране появляется отчет:
   - 4 скора (Performance / Accessibility / Best Practices / SEO)
   - Core Web Vitals (LCP / CLS / INP) с расшифровкой
   - Топ-5 проблем с приоритизацией
   - Что это значит для бизнеса (потеря позиций, конверсии)
6. Крупный CTA: «Хотите это исправить? Vibecraft сделает за 1-2 дня.
   Промокод AUDIT15 — скидка 15% на первый заказ.»
7. Email клиента сохраняется в Supabase
8. Resend отправляет welcome-email:
   «Спасибо за аудит. Промокод AUDIT15 действителен 14 дней.
   Если нужна помощь — напишите в Telegram @vibecraft_kz»
```

### Rate limiting

- Не более 3 запросов в час с одного IP
- Не более 1 запроса в день на один email
- Защита от ботов: простая captcha или honeypot

## 3. Что показывать в отчете на экране

Отчет рендерится как React-компонент на странице, не скачивается как PDF.

### Блок 1: Краткая сводка

4 круговых индикатора (SVG) с числами 0-100:
- Performance (mobile)
- Accessibility
- Best Practices
- SEO

Цвета: 0-49 красный (#EF4444), 50-89 желтый (#F59E0B), 90-100 зеленый (#10B981).
Показывать для mobile (основной) и desktop (вторичный, мелким).

### Блок 2: Core Web Vitals

3 метрики с пояснениями:
- **LCP** (Largest Contentful Paint) — «Сколько ждет пользователь пока страница загрузится»
  - Хорошо: < 2.5с / Средне: 2.5-4с / Плохо: > 4с
- **CLS** (Cumulative Layout Shift) — «Прыгает ли контент на странице»
  - Хорошо: < 0.1 / Средне: 0.1-0.25 / Плохо: > 0.25
- **INP** (Interaction to Next Paint) — «Как быстро сайт реагирует на нажатия»
  - Хорошо: < 200мс / Средне: 200-500мс / Плохо: > 500мс

Для каждой метрики: текущее значение + «норма Google» + статус (хорошо/средне/плохо).

### Блок 3: Топ-5 проблем

Из PSI-данных раздел «Opportunities» — 5 самых эффективных исправлений.
Для каждого:
- Название проблемы на русском (не «Reduce unused JavaScript» а «На сайте загружается лишний код»)
- Потенциальная экономия в мс или КБ
- Приоритет: Критично / Важно / Низкий

### Блок 4: CTA (главный)

Крупный блок с градиентом Vibecraft (#8B5CF6 → #EC4899):
- Заголовок: «Хотите это исправить?»
- Текст: «Vibecraft устранит найденные проблемы за 1-2 рабочих дня. Промокод AUDIT15 — скидка 15%.»
- Кнопка: «Написать в Telegram» → ведет на t.me/vibecraft_kz
- Или: «Оставить заявку» → mailto/форма

## 4. Технический стек

| Компонент | Технология |
|---|---|
| Страница | vibecraft.kz/audit (Next.js, часть существующего проекта) |
| PSI запрос | Next.js API Route `/api/audit` → Google PSI API (синхронно, 10-30 сек) |
| Email логирование | Supabase таблица `audit_leads` (email, url, timestamp, scores) |
| Welcome email | Resend (один шаблон, отправка после аудита) |
| UI отчета | React-компоненты с SVG-графиками (не PDF, не скачиваемый файл) |
| Rate limiting | In-memory + Supabase check по email |
| Аналитика | Vercel Analytics + GA4 event tracking |

### API Route: /api/audit

```
POST /api/audit
Body: { url: string, email: string }

1. Валидация URL и email
2. Rate limit check (IP + email)
3. Вызов Google PSI API (mobile + desktop)
4. Сохранить email + URL + scores в Supabase
5. Отправить welcome-email через Resend
6. Вернуть JSON с результатами PSI
```

PSI API вызывается синхронно. 10-30 секунд — приемлемо, клиент видит индикатор загрузки. Очередь задач (Inngest) НЕ нужна.

### Supabase таблица audit_leads

```sql
CREATE TABLE audit_leads (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  email TEXT NOT NULL,
  url TEXT NOT NULL,
  scores_mobile JSONB,
  scores_desktop JSONB,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_audit_leads_email ON audit_leads(email);
```

Не хранить полные результаты PSI — только скоры и email. Для воронки этого достаточно.

## 5. Воронка в Vibecraft

### Точки конверсии

1. **CTA на экране отчета** — «Хотите это исправить? → Написать в Telegram»
2. **Welcome email** — промокод AUDIT15, срок действия 14 дней
3. **Follow-up email (через 3 дня)** — «Как дела с сайтом? Нужна помощь?»
4. **Карточка на vibecraft.kz** — в секции Services: «Аудит скорости сайта онлайн бесплатно» → /audit
5. **Footer vibecraft.kz** — ссылка на /audit

### Промокод AUDIT15

- 15% скидку на первый заказ в Vibecraft
- Привязка: email из аудита → CRM Vibecraft
- Срок действия: 14 дней
- Трекинг: UTM-метка в ссылке из email

### KPI воронки

| Метрика | Цель (через 2 мес) |
|---|---|
| Использований /audit в месяц | 100+ |
| Конверсия «использовал → оставил email» | 60-70% |
| Конверсия «email → заявка в Vibecraft» | 5-10% |
| Заявки в Vibecraft через инструмент | 3-5 в месяц |

## 6. План реализации

| День | Задачи |
|---|---|
| День 1 | Создать страницу /audit в vibecraft.kz. Форма ввода URL + email. Индикатор загрузки. API Route для PSI. |
| День 2 | React-компоненты отчета: 4 скора (SVG-круги), Core Web Vitals, Топ-5 проблем. Перевод ключей PSI на русский. |
| День 3 | CTA-блок с градиентом Vibecraft. Supabase таблица audit_leads. Rate limiting. |
| День 4 | Resend: welcome-email шаблон + отправка. Follow-up email через 3 дня (cron или Resend scheduled). GA4 event tracking. |
| День 5 | Тестирование на 10 разных сайтах. Фикс edge cases (недоступный URL, таймаут PSI, невалидный email). Деплой. |

**Суммарно: 5 рабочих дней.**

### Что НЕ делаем

- PDF-генерация
- Личный кабинет / авторизация
- Платежи
- Подписки / мониторинг
- Отдельный домен/поддомен
- Очередь задач
- История аудитов
