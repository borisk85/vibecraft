## Выводы для стратегии

**1. База несравнимо сильнее VELA.** 2.18M SMB, +5.2%, 733 млрд ₸ на разработку ПО. При чеке 150k-2M₸ тебе нужно **3-4 клиента/мес** для $3-5K. Это реальный соло-доход, а не $9-подписки.

**2. Главная проблема — claim «в 2-3× дешевле студий» завышен.**
- Держится ТОЛЬКО на MVP (1.2M vs 2M = −40%).
- По ботам тебя ПОДРЕЗАЮТ (60-95k vs твои 150k).
- По сайтам и мобилкам ты на уровне рынка.
- **Не веди маркетинг на «дешевле» — проиграешь botproject.kz по цене.** Веди на СКОРОСТЬ (1-2 недели vs месяцы) + AI-возможности, которых у commodity-шопов нет + качество Claude.

**3. Ниша НЕ пустая.** 100+ AI-компаний, бот-студии, веб-студии. «AI-агентство» само по себе не дифференциатор. Дифференциация — конкретные AI-функции (AI-чат-консультант, калькулятор, мини-аудит), которые студии не делают быстро, + скорость.

**4. AI-спрос у SMB ранний.** Не продавай «AI-интеграцию» (для SMB пустой звук) — продавай ИСХОД: «бот принимает заказы», «автоматизация экономит N часов». РФ-прецедент (53% SMB в маркетинге) показывает: волна идет, ты рано, но не слишком.

**5. Каналы (из твоего marketing-plan + рынок):** white-label субподряд для digital-агентств (теплый спрос, проверенная потребность) + сарафан + SEO-лендинги под buyer-intent КЗ-запросы. Холод/платное — позже.

**Вердикт:** рынок есть, база большая, чеки B2B — $500-2000/мес достижимо при 2-4 клиентах. Но позиционирование «дешевле» — ложное и опасное; переключай на «быстрее + AI-функции + качество». Ниша конкурентная, бери исходом и скоростью, не ценой.
