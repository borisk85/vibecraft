# Vibecraft — SEO keyword research (июль 2026)

**Дата:** 2026-07-07
**Метод:** XMLRiver Wordstat (частотность RU, прокси КЗ-спроса) + WebSearch SERP-конкуренция. Только КЗ-таргет, без EN-рынка.

## Главные выводы
- **Гео-ключи («казахстан»/«алматы») почти пустые** (0-11/мес) — Wordstat меряет в основном РФ, казахстанцы ищут те же ключи без гео-слова. Таргетируем общие RU-ключи + КЗ-якоря ВНУТРИ статьи (Kaspi, Halyk, ₸, Алматы, ТОО, ЭСФ, 2ГИС).
- **SERP по КЗ-B2B занят коммерческими страницами услуг локальных студий** (splus.kz, onesystem, applications.kz, A-LUX, osn.kz, chatbot.kz), а не медиа-листиклами. По ценовым — Forbes.kz + OSN-блог.
- **Head-коммерческие ключи блогом НЕ взять** (страницы услуг конкурентов + низкий DR vibecraft) → закрываем страницами услуг на сайте. **Блог бьет long-tail** — там топ-3/5 реально за 2-4 мес.
- **Валюта: держим ₸, КЗ-фокус.** Домен .kz на Google.ru проигрывает .ru-доменам, за РФ не гонимся. $ отпугивает B2B-СМБ.

## ПЛАН СТАТЕЙ ПОД БЫСТРЫЙ ТРАФИК (порядок написания)

Логика: низкий DR → head-ключи в лоб не берем (топ забит страницами студий, трафик через месяцы). Быстрый трафик = long-tail с реальным спросом и слабым SERP, топ-3 за 3-6 недель.

### Блок A — писать ПЕРВЫМИ (топ 3-6 недель, спрос есть, конкуренции нет)
1. **«Приложение для доставки: сколько стоит и что входит»** — приложение для доставки (6212) + для магазина (398) + сколько стоит разработка приложения (342). Почему: жирный спрос, SERP без гигантов, buyer-intent.
2. **«Автоматизация документооборота / склада для малого бизнеса»** — документооборота (2428) + склада (2352) + отчетов (2849). Почему: 2400+/мес, SERP не забит медиа, конкретная боль.
3. **«Как внедрить ИИ в бизнес в Казахстане»** — внедрение ии в бизнес (819) + ии для автоматизации (1667) + как внедрить (91). Почему: ранний рынок, слабый SERP, прямое УТП AI-студии.
4. **«Приложение и чат-бот для записи клиентов: салон, клиника, СТО»** — приложение для записи (1832) + бот для записи (157) + телеграм бот для записи (83). Почему: спрос 1832, спец-страниц нет, точный intent.

### Блок B — отраслевые мини-статьи (топ-1 сразу, трафика мало, лиды горячие)
5. **«Чат-бот для стоматологии/салона/магазина/ресторана»** — стоматология (29), салон (28), клиники (22), интернет магазин (16), ресторан (9). Почему: конкуренция нулевая → топ-1 моментально; кто ищет = готовый клиент.

### Блок C — ценовые (спрос огромный, топ-5, дольше — параллельно)
6. **«Сколько стоит приложение/сайт/чат-бот в Казахстане 2026»** — сколько стоит сайт (24618) + приложение (342) + чат бот (360) + разработка сайта цена (618). Почему: гигантский спрос, но топ = Forbes.kz/OSN → топ-5 да, топ-3 долго.
7. **«Автоматизация отдела продаж и CRM»** — автоматизация продаж (1977) + crm малый бизнес (768) + amocrm (681). Почему: 1977, средний SERP, топ-5.
